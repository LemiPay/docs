# Casos de Uso
Casos de uso para los distintos Actores

## Logged User

- Unirse a un grupo, si se lo invita
- Ver los grupos que forma parte y sus detalles  
- Fondear la tesorería del grupo (ya sea crypto o algún servicio on-ramp)  
- Proponer un gasto  
- Aprobar un gasto  
- Proponer Inversión  
- Aprobar Inversión  
- Revisar estado de las inversiones, ganancias, pérdidas y estrategias utilizadas  
- Irse del grupo  

---

## Group Owner

- Crear nuevo grupo  
- disolver el grupo  
- Definir umbral de aprobación para gastos/inversiones etc (cuantos votos hay que tener)  
- Agregar o remover gente del grupo  
- Todo lo que puede hacer un Logged User  

---

## Unloggued User

- Sign in usando wallets  
- Sign in using mail (abstracted from account)  
- Review Docs
